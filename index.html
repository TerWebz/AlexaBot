Big Sur Discord Bot

Big Sur is a fun moderation bot for your Discord server!

Support Invite

Commands


bs!Help               This will give you links and information.
bs!Kick [user]        Kicks the mentioned user
bs!Ban [user]         Bans the mentioned user
bs!Unban [user]       Unbans the mentioned user
bs!Purge [number]     Bulk deletes specified number of messages
bs!Google             Searches for your query
bs!YouTube            Searches for your query
bs!Weather            Gives you the weather based on country/state.
bs!Trash [user]       Turns the specified user into trash.
bs!HDDVD [url]        Overlays an image with the HD-DVD template
bs!Lock               Locks the current channel. (requires perms)
bs!Unlock             Unlocks the current channel. (requires perms)
bs!Roblox [user]      Grabs a Roblox user's profile!
bs!Corona             Gives you COVID-19 info based on country/state.
bs!Mute [user]        Mutes the specified user. (requires perms)
bs!Unmute [user]      Umutes the specified user. (requires perms)
bs!AFK [reason]       Turns you AFK.
bs!Ping               Pings Discord servers.
bs!RiiTag [user]      Gets a user's RiiTag
bs!Meme               Sends a random meme.
bs!CoinFlip           Flip a coin!
bs!ChangeSetting      Changes settings, list TBA, case-sensitive. (Try a!ChangeSetting welcomemsg enable/disable)
bs!Play               Plays a song. - Powered by YouTube
bs!Join               Joins your voice channel.
bs!8Ball              Magic 8-Ball!
bs!Hack               "Hacks" a user.
bs!whoami              Who am I?!
bs!sudo                Sudo some commands
bs!Calculator          Now you can cheat in class!
