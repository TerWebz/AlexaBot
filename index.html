Alexa Discord Bot

Alexa is a free fun moderation bot for your Discord server!

Support Commands Invite

Commands


a!Help               This will give you links and information.
a!Kick [user]        Kicks the mentioned user
a!Ban [user]         Bans the mentioned user
a!Unban [user]       Unbans the mentioned user
a!Purge [number]     Bulk deletes specified number of messages
a!Google             Searches for your query
a!YouTube            Searches for your query
a!Weather            Gives you the weather based on country/state.
a!Trash [user]       Turns the specified user into trash.
a!HDDVD [url]        Overlays an image with the HD-DVD template
a!Lock               Locks the current channel. (requires perms)
a!Unlock             Unlocks the current channel. (requires perms)
a!Roblox [user]      Grabs a Roblox user's profile!
a!Corona             Gives you COVID-19 info based on country/state.
a!Mute [user]        Mutes the specified user. (requires perms)
a!Unmute [user]      Umutes the specified user. (requires perms)
a!AFK [reason]       Turns you AFK.
a!Ping               Pings Discord servers.
a!RiiTag [user]      Gets a user's RiiTag
a!Meme               Sends a random meme.
